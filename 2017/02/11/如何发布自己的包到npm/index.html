如何引入第三方包

模块分类：

  • 第三方模块： 引入：require(模块名)
  • 文件模块： 引入：require(./模块名)
  • 核心模块： 不需要下载，node自带,例如：util模块

查看包的发布信息 npm info wplpackage

node_modules文件夹下的模块称之为第三方模块
