轻松掌握js继承

发表于 2017-02-11 | 阅读次数 |

这是我学习原生js继承的一些核心思想，总共有六种方式

原型继承

核心：原型继承:子类的原型指向了父类的一个实例，父类的公有的和私有的方法都是子类的公有的方法

call方法继承

在子类的构造函数中执行，C.call(this) 父类.call(this); 其实就是把父类私有的添加到子类私有的方法。

冒充对象继承

冒充对象继承：在子类构造函数中遍历父类实例，把父类的私有和公有都变成子类的私有。

组合继承

无论是私有的还是共有的全部都能够实现继承，原来是私有的现在仍然通过call继承，仍然是私有的，原来是公有的，通过原型继承是公有的

原型链继承(中间件继承)-不兼容

让子类的原型的proto属性，本来应该指向Object.prototype;现在让他指向父类的原型，父类的公有属性变成子类的公有属性。(这个可以联想到你要从上海到北京，你可以中间途经一个中转站到达北京)
